# Web Helper

A side-panel workbench for capturing web pages and selections as clean markdown —
edit it locally, then copy or export it to the AI assistant of your choice.

No accounts, no API keys, nothing leaves your machine.

- [Privacy Policy](privacy-policy.html)
- [Source code on GitHub](https://github.com/saarsg/web-helper)

Built for research and personal use. Open source under the MIT license.
